Tag H1 . . . H6

As tags H1, H2, H3, H4, H5 e H6 são utilizadas para definir um titulo na página, essa tag vai até o H6! Quanto maior o número, menor o titulo. Ou seja o titulo maior que você pode ter, é utilizando a tag H1 e o menor titulo é o H6.

Exemplo:

TEXTO COM H1

TEXTO COM H2

TEXTO COM H3

TEXTO COM H4

TEXTO COM H5
TEXTO COM H6

Tag Ol

Essa tag se chama Lista Ordenada ou Ordered List, significa que ela é responsavel por ordenar numericamente os itens de uma lista. Ela é uma tag mãe, pois dentro dela costumamos colocar uma outra tag chamada "li ou Item List".
Exemplo:

  1. Item 1
  2. Item 2
  3. Item 3

Voltar

Tag Ul

Essa é uma tag parecida com a anterior, o que muda é que no lugar dos números que ordenam a lista, nessa tag é usado um pontinho para identificar o item da lista. Essa tag tambem é uma tag mãe na qual dentro dela vai o Li/Item List
Exemplo:

  • Item 1
  • Item 2
  • Item 3
